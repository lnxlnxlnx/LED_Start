# 02. 常用容器与使用套路

这一份看“怎么选容器、怎么记 API”。

## 1. 先记住：`XVector` 是核心

在当前这套实现里，`XVector` 是最值得先吃透的容器，因为很多别的容器都是在它基础上复用出来的。

- `XStack`：本质上复用 `XVector` 的尾插、尾删、取尾
- `XPriorityQueue`：底层也是 `XVector`
- 一部分算法接口也是围绕顺序存储结构设计的

所以你真正先学会 `XVector`，后面会轻松很多。

## 2. XVector 的高频 API

### 创建与销毁

```c
XVector* vec = XVector_Create(int);
XVector_delete_base(vec);
```

### 插入

```c
XVector_push_back_base(vec, &value);
XVector_push_front_base(vec, &value);
XVector_insert(vec, index, &value);
```

### 删除

```c
XVector_pop_back_base(vec);
XVector_pop_front_base(vec);
XVector_remove_base(vec, index, n);
```

### 访问

```c
XVector_at_base(vec, index);
XVector_front_base(vec);
XVector_back_base(vec);
XVector_size_base(vec);
XVector_capacity_base(vec);
```

### 查找与排序

```c
XVector_find_base(vec, &value, &it);
XVector_sort_base(vec, XSORT_ASC);
```

## 3. `*_base` 和大写宏的区别

这个库很喜欢同时提供两套接口。

### `*_base`

这是底层通用接口，通常吃的是 `void*`。

```c
int x = 123;
XVector_push_back_base(vec, &x);
```

### 大写宏

这是“类型友好一点”的封装，会帮你生成临时变量。

```c
XVector_Push_Back_Base(vec, int, 123);
```

所以可以把它理解成：大写宏更像“模板语法糖”，`_base` 才是真正干活的接口。

## 4. `move` 版本是什么意思

你会看到很多名字长这样：

- `XVector_push_back_base`
- `XVector_push_back_move_base`

它表达的是两种语义：

- `copy`：把你的数据复制一份进容器
- `move`：把资源所有权交给容器

但要注意，C 里没有真正自动的移动语义。这里的“move”能否成立，取决于你有没有给容器设置好 `m_dataMoveMethod`。

## 5. 迭代器怎么用

这个库提供了伪 STL 风格的遍历方式：

```c
for_each_iterator(vec, XVector, it)
{
    int* value = (int*)XVector_iterator_data(&it);
    printf("%d\r\n", *value);
}
```

这套写法底层依赖：

- `XVector_begin`
- `XVector_end`
- `XVector_iterator_add`
- `XVector_iterator_equality`

本质上还是 C 宏展开，不是语言级迭代器。

## 6. XStack：最容易理解的派生容器

`XStack` 基本可以理解成“给 `XVector` 换了一层名字”。

```c
XStack* stack = XStack_Create(int);
XStack_Push_Base(stack, int, 10);
XStack_Push_Base(stack, int, 20);

int top = XStack_Top_Base(stack, int);
XStack_pop_base(stack);
XStack_delete_base(stack);
```

它的设计思想很直接：

- 压栈 = `XVector_push_back`
- 出栈 = `XVector_pop_back`
- 栈顶 = `XVector_back`

## 7. XQueue：底层偏链表

`XQueue` 在当前实现里是基于 `XListSLinked` 的，所以它偏向 FIFO 场景。

```c
XQueue* q = XQueue_Create(int);
XQueue_Push_Base(q, int, 1);
XQueue_Push_Base(q, int, 2);

int front = XQueue_Top_Base(q, int);
XQueue_pop_base(q);
XQueue_delete_base(q);
```

学习这个容器时，重点不是 API，而是看它怎么“复用链表的尾插 + 头删”实现队列。

## 8. XPriorityQueue：底层偏堆

它的底层是 `XVector`，额外带一个：

- 比较函数 `XCompare`
- 排序方向 `XSortOrder`

你至少要这样创建：

```c
XPriorityQueue* pq = XPriorityQueue_Create(int, (XCompare)XCompare_int, XSORT_ASC);
```

然后再 `push / top / pop`。

## 9. 学习路线建议

建议按这个顺序练：

1. `XVector`
2. `XStack`
3. `XQueue`
4. `XPriorityQueue`
5. 再去看 `XListSLinked` / `XListDLinked`

因为前四个已经足够把这套库的大部分设计套路看懂。
