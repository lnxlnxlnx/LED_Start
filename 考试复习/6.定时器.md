# STM32F103 定时器TIM 全套简答题+编程解答
## 1. STM32F103x 定时器结构与工作原理
### 硬件核心结构（通用定时器TIM2/TIM3/TIM4）
1. **时钟预分频器PSC**：16位预分频，对输入时钟分频，分频系数 `PSC+1`，降低计数器计数频率；
2. **16位自动装载计数器CNT**：核心计数寄存器，向上/向下/中心对齐计数；
3. **自动重装载寄存器ARR**：计数溢出阈值，CNT计数到ARR值触发更新事件，CNT清零/反向；
4. **捕获比较单元CCRx（4路：CCR1~CCR4）**：输入捕获、输出比较/PWM生成；
5. **控制寄存器组**：CR1/CR2、SMCR（从模式控制器）、IER中断使能；
6. **外部输入通道**：TI1~TI4（引脚捕获）、ETR外部触发时钟输入。

### 工作原理
外部/内部时钟经过PSC预分频后驱动CNT计数器累加/递减；CNT计数值与ARR、CCR寄存器持续比较：
- 定时模式：CNT溢出触发更新中断，实现固定时长定时；
- 输入捕获：引脚电平边沿触发，锁存当前CNT值到CCRx，测量脉冲宽度/频率；
- 输出比较/PWM：CNT与CCRx匹配时翻转引脚电平，生成PWM波形；
- 从模式：由外部TI/ETR信号触发计数、复位、门控，实现定时器同步。

## 2. 定时器主要功能
1. **基本定时延时**：溢出中断实现毫秒/秒级定时，周期性任务调度；
2. **输入捕获**：检测引脚上升/下降沿，测量脉冲周期、高电平宽度、外部信号频率；
3. **输出比较**：匹配时输出高低电平，产生精准延时脉冲；
4. **PWM脉冲宽度调制**：生成可调占空比PWM，驱动电机、LED调光；
5. **从模式同步**：多定时器级联、外部触发启动/复位计数器；
6. **正交编码器接口**：配合编码器测速、定位；
7. **外部时钟计数**：对外部脉冲计数，统计脉冲个数。

## 3. 通用定时器计数器三种计数方式
### （1）向上计数（递增计数）
CNT从0开始递增，计数到`ARR`值时产生更新事件，CNT清零重新从0开始计数，最常用定时模式。
### （2）向下计数（递减计数）
CNT从ARR初始值开始递减，减到0时触发更新事件，重载ARR值继续递减计数。
### （3）中央对齐模式（双向三角计数）
CNT从0递增到ARR → 触发更新，改为递减计数到0 → 再次更新、重新递增；
多用于对称PWM生成，电机驱动，上下溢出都会产生中断。

## 4. 定时器四大时钟源
1. **内部时钟CK_INT（默认）**：APB预分频后内部总线时钟，定时器主时钟，定时场景首选；
2. **TIx外部输入时钟**：TI1/TI2/TI3/TI4引脚边沿作为计数时钟，引脚脉冲触发CNT计数；
3. **ETR外部触发时钟**：ETR专用外部引脚高速外部时钟，可预分频、极性反转；
4. **内部触发ITRx**：多个定时器互联，主定时器触发从定时器同步计数，多TIM级联同步。

## 5. 配置：向下计数、TI2输入端下降沿计数
1. 开启TIMx对应APB总线时钟，开启TI2对应GPIO端口时钟，GPIO配置为浮空输入；
2. 配置TIM从模式控制器SMCR：选择**TI2触发输入**，触发极性选择**下降沿**；
3. TIM_CR1寄存器配置计数器模式：向下计数（DIR=1）；
4. 配置TI2滤波器、预分频（可选，消抖滤波）；
5. 使能计数器CEN位（CR1.CEN=1），TI2下降沿到来时CNT递减1；
6. 可选开启更新中断，计数到0进入中断。

## 6. 配置ETR上升沿触发、向上计数
1. 开启TIMx时钟、ETR对应GPIO时钟，GPIO设为浮空输入；
2. SMCR寄存器配置：时钟源选择ETR外部触发，ETR极性设为上升沿有效；
3. 配置ETR预分频、数字滤波（抗干扰）；
4. CR1寄存器DIR=0，设置计数器**向上计数**；
5. 从模式配置为外部时钟模式1，由ETR边沿驱动计数；
6. 置位CR1.CEN使能定时器，每一次ETR上升沿CNT+1。

## 7. TI1下降沿捕获，将CNT值锁存到CCR1配置步骤
1. 开启TIMx时钟、TI1对应GPIO时钟，GPIO配置浮空输入；
2. 配置TIM通道1为**输入捕获模式**，映射TI1输入；
3. 捕获极性配置：下降沿触发捕获（CCER.CC1P=1）；
4. 配置输入滤波、预分频，消除引脚抖动；
5. 使能通道1捕获允许（CCER.CC1E=1）；
6. 使能定时器计数器CEN；TI1下降沿触发硬件自动锁存当前CNT值到TIMx_CCR1，可开启捕获中断读取数值。

## 8. 定时器0.5s中断，RGB LED红→绿→蓝循环（标准库TIM2中断示例）
### 硬件定义
- TIM2 基础定时中断，72MHz系统时钟，预分频PSC=7199，ARR=4999，定时0.5s
- 红灯PB0、绿灯PB1、蓝灯PB2，推挽输出，低电平点亮
```c
#include "stm32f10x.h"

// GPIO初始化 RGB灯
void RGB_GPIO_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct;
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOB, ENABLE);
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_2MHz;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    GPIO_SetBits(GPIOB, GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2); // 初始全灭
}

// TIM2 0.5s中断初始化
void TIM2_Init_500ms(void)
{
    TIM_TimeBaseInitTypeDef  TIM_TimeBaseStruct;
    NVIC_InitTypeDef NVIC_InitStruct;
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM2, ENABLE); // TIM2挂载APB1

    // 72MHz/(7199+1)/(4999+1) = 72000000/7200/5000 = 2Hz → 0.5s
    TIM_TimeBaseStruct.TIM_Prescaler = 7199;
    TIM_TimeBaseStruct.TIM_Period = 4999;
    TIM_TimeBaseStruct.TIM_ClockDivision = TIM_CKD_DIV1;
    TIM_TimeBaseStruct.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM2, &TIM_TimeBaseStruct);

    TIM_ITConfig(TIM2, TIM_IT_Update, ENABLE); // 使能更新中断
    // 中断优先级配置
    NVIC_InitStruct.NVIC_IRQChannel = TIM2_IRQn;
    NVIC_InitStruct.NVIC_IRQChannelPreemptionPriority = 1;
    NVIC_InitStruct.NVIC_IRQChannelSubPriority = 1;
    NVIC_InitStruct.NVIC_IRQChannelCmd = ENABLE;
    NVIC_Init(&NVIC_InitStruct);

    TIM_Cmd(TIM2, ENABLE); // 开启定时器
}

uint8_t led_state = 0; //0红 1绿 2蓝
void TIM2_IRQHandler(void)
{
    if(TIM_GetITStatus(TIM2, TIM_IT_Update) != RESET)
    {
        TIM_ClearITPendingBit(TIM2, TIM_IT_Update);
        GPIO_SetBits(GPIOB, GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2); //熄灭所有
        switch(led_state)
        {
            case 0: GPIO_ResetBits(GPIOB, GPIO_Pin_0); break;
            case 1: GPIO_ResetBits(GPIOB, GPIO_Pin_1); break;
            case 2: GPIO_ResetBits(GPIOB, GPIO_Pin_2); break;
        }
        led_state = (led_state + 1) % 3; //三色循环
    }
}

int main(void)
{
    RGB_GPIO_Init();
    TIM2_Init_500ms();
    while(1)
    {
        // 定时器中断自动翻转，主循环空循环
    }
}
```
### 定时参数计算
$f_{cnt}=72MHz/(PSC+1)=72M/7200=10kHz$
溢出周期 $T=(ARR+1)/f_{cnt}=5000/10000=0.5\ \text{s}$，满足题目要求。
