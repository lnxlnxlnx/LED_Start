# STM32 DMA 五道简答题完整解答
## 1. DMA 概念与作用
### 概念
DMA（Direct Memory Access，直接存储器访问），是芯片内部独立于CPU的硬件数据搬运控制器，允许**外设与内存、内存与内存、外设与外设**之间直接传输数据，全程无需CPU介入数据搬运。

### 核心作用
1. **解放CPU**：数据块搬运由DMA硬件独立完成，CPU无需反复读写寄存器，可并行执行运算、逻辑任务，提升系统整体效率；
2. **高速批量传输**：硬件总线直接传输，速率远高于CPU软件循环读写，适合ADC采样、串口大批量收发、SPI高速数据搬运；
3. **降低功耗**：CPU可进入休眠模式，由DMA完成数据交互，低功耗场景优势明显；
4. **减轻总线压力**：硬件总线仲裁机制，固定优先级通道传输，数据传输稳定无软件延时。

## 2. DMA 三种传输方式
STM32F103 DMA共3种工作传输模式，由配置寄存器控制：
### （1）单次传输模式（正常模式）
设定传输N个数据，数据全部传输完成后，DMA通道自动关闭，停止传输；需要重新软件使能才能再次传输。适合一次性批量数据搬运。
### （2）循环传输模式（连续循环模式）
传输计数到0后，自动重装初始传输长度，循环往复持续传输，通道不会关闭。典型场景：ADC连续采样、音频数据流不间断收发。
### （3）内存到内存传输模式
仅DMA1支持，无需外设触发，软件启动后直接完成片内SRAM两块区域数据搬家，完全由DMA自主搬运，不依赖外设硬件请求。

## 3. STM32F103 两个DMA控制器：DMA1、DMA2 划分与特点
整体架构：DMA1（7通道）、DMA2（5通道），合计12通道；挂载不同总线，外设绑定不同，权限不同。
### （1）DMA1
- 挂载总线：**APB1总线**
- 通道数量：7个独立通道
- 绑定外设：低速/中速APB1外设：USART2/3、SPI2、I2C、TIM2/3/4、ADC等
- 独有特性：**支持内存→内存直接传输**，DMA2不支持该功能；
- 适用场景：串口收发、常规ADC采样、定时器数据搬运。

### （2）DMA2
- 挂载总线：**APB2总线**
- 通道数量：5个独立通道
- 绑定外设：高速APB2外设：USART1、SPI1、TIM1、高级定时器、外部高速外设
- 限制：**不支持内存到内存传输**，仅能外设触发传输
- 适用场景：高速串口、PWM波形数据、高速SPI通信、电机控制外设数据搬运

### 核心区分总结
1. DMA1：APB1外设、7通道、支持MEM→MEM；
2. DMA2：APB2外设、5通道、无内存互传，适配高速外设。

## 4. STM32F103 DMA 请求/应答握手协议（2种）
### （1）硬件请求模式（外设触发，最常用）
外设产生数据收发请求（如USART接收非空、ADC转换完成），向DMA发送硬件传输请求；DMA获得总线使用权后完成传输，传输完成后外设清除请求标志，属于**事件触发握手**，分两种信号：
1. 单次请求：外设产生一次请求，DMA传输1个数据；
2. 块请求：外设持续请求，DMA批量传输预设长度数据。

### （2）软件触发模式（仅DMA1内存互传）
CPU软件置位传输启动位，直接发起DMA传输，无外设硬件握手，纯软件命令启动内存块搬运，无需外部应答信号。

补充：通道优先级仲裁：多个通道同时请求时，硬件按软件配置的优先级（高/中/低）固定仲裁，高优先级通道优先占用总线，属于内部总线应答机制。

## 5. DMA 通用配置完整步骤（标准库）
1. **开启总线时钟**：开启对应DMA外设时钟（DMA1/DMA2），同时开启触发该DMA的外设时钟（如USART、ADC）；
2. **DMA初始化结构体配置**
    - 选择通道、传输方向：外设→内存 / 内存→外设 / 内存→内存；
    - 配置外设地址、内存地址；
    - 传输数据长度、数据位宽（8/16/32bit）；
    - 增量模式：外设地址是否递增、内存地址是否递增；
    - 传输模式：单次/循环模式；
    - 通道优先级；
3. 调用`DMA_Init()`加载基础配置；
4. 可选中断配置：开启传输完成/半传输/错误中断，配置NVIC优先级，用于传输结束通知CPU；
5. 使能DMA通道：`DMA_Cmd(DMA_Channelx, ENABLE)`；
6. **配套外设配置**：在外设中开启DMA请求使能（如USART_DMACmd、ADC_DMACmd），让外设可以触发DMA搬运。

### 背诵精简考点
1. DMA核心：硬件自主搬数，解放CPU；
2. 传输模式：单次、循环、内存到内存；
3. DMA1：APB1、7通道、支持内存互传；DMA2：APB2、5通道、高速外设专用；
4. 触发方式：外设硬件请求、软件触发；
5. 配置核心：开时钟→配地址/方向/增量/长度→开通道→外设开启DMA触发。
