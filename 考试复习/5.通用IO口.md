# STM32F103 GPIO 知识点+编程题完整解答
## 1. GPIO 操作与配置整体流程
### 一、操作方式
1. **寄存器直接操作**：直接对端口配置寄存器`CRL/CRH`、输入数据寄存器`IDR`、输出数据寄存器`ODR`、置位复位寄存器`BSRR`、复位寄存器`BRR`读写，底层高效；
2. **标准库(STD库)函数操作**：调用库封装函数`GPIO_Init()`、`GPIO_WriteBit()`、`GPIO_ReadInputDataBit()`等，可读性强；
3. **HAL库操作**：HAL_GPIO_Init、HAL_GPIO_WritePin、HAL_GPIO_ReadPin，现代主流开发方式。

### 二、标准配置核心步骤
1. 开启GPIO端口对应**APB2时钟**（GPIOA~GPIOG都挂载APB2总线，时钟默认关闭，必须先开时钟）；
2. 定义`GPIO_InitTypeDef`结构体，配置：端口引脚、工作模式、输出速率；
3. 调用`GPIO_Init()`完成硬件初始化；
4. 输入：读`IDR`寄存器电平；输出：写`ODR/BSRR/BRR`控制高低电平。

## 2. STM32F103 GPIO 8种工作模式（4大类）
|模式分类|具体模式|功能说明|
| ---- | ---- | ---- |
|输入类|浮空输入（GPIO_Mode_IN_FLOATING）|默认上电状态，无上下拉，外部电平直接读取，适合按键外接下拉|
|输入类|上拉输入（GPIO_Mode_IPU）|内部接上拉电阻，无外部信号时稳定高电平|
|输入类|下拉输入（GPIO_Mode_IPD）|内部接下拉电阻，无外部信号时稳定低电平|
|输出类|通用推挽输出（GPIO_Mode_Out_PP）|推拉驱动，高低电平驱动能力强，驱动LED、外设|
|输出类|通用开漏输出（GPIO_Mode_Out_OD）|仅拉低能力，高电平需外部上拉，适合线与通信|
|复用输出|复用推挽输出（GPIO_Mode_AF_PP）|引脚复用外设（USART、SPI、TIM）推挽驱动|
|复用输出|复用开漏输出（GPIO_Mode_AF_OD）|外设复用开漏，I2C标准配置模式|
|模拟输入（ADC专用）|GPIO_Mode_AIN|关闭数字缓冲，直接接入ADC模拟采集，禁止数字读写|

## 3. STM32F103 GPIO 输出速度档位（3种）
输出速度即引脚翻转最大频率，控制压摆率、EMI干扰：
1. **2MHz**：低速，低功耗、低电磁干扰，普通LED、低速外设首选
2. **10MHz**：中速，常规通用外设
3. **50MHz**：高速，高速通信（SPI、定时器PWM），功耗与干扰更大

## 4. 输出高低电平由哪些寄存器决定
1. **ODR（输出数据寄存器）**：每一位对应引脚，写1输出高电平、写0输出低电平，可读可写；
2. **BSRR（置位复位寄存器）**：只写寄存器，低16位置1对应引脚置高，高16位置1对应引脚清零，**原子操作，无读改写冲突**，推荐优先使用；
3. **BRR（复位寄存器）**：只写寄存器，置1对应引脚强制拉低，仅清零专用。
> 补充：推挽/开漏模式决定硬件驱动结构，最终电平由以上三个输出寄存器逻辑决定。

## 5. GPIO 标准初始化流程（STD库）
1. **总线时钟使能**：调用`RCC_APB2PeriphClockCmd()`开启目标GPIO端口时钟（必须第一步，时钟未开启寄存器无响应）；
2. **结构体初始化**：定义`GPIO_InitTypeDef GPIO_InitStruct`，配置`GPIO_Pin`引脚号、`GPIO_Mode`工作模式、`GPIO_Speed`输出速度（输入模式无需配置速度）；
3. **初始化函数载入配置**：`GPIO_Init(GPIOx, &GPIO_InitStruct)`，把配置写入硬件CRL/CRH寄存器；
4. **可选初始电平设定**：初始化后用`GPIO_WriteBit`/BSRR设置默认初始电平，避免上电电平抖动。

## 6. 程序：GPIOB.0置1、GPIOB.1清零（STD库示例）
```c
#include "stm32f10x_gpio.h"
#include "stm32f10x_rcc.h"

void GPIO_Config_SetClear(void)
{
    GPIO_InitTypeDef GPIO_InitStruct;
    // 1. 开启GPIOB时钟
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOB, ENABLE);
    
    // 2. 配置PB0、PB1为推挽输出 10MHz
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_0 | GPIO_Pin_1;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_10MHz;
    GPIO_Init(GPIOB, &GPIO_InitStruct);

    // 方式1：BSRR原子置位清零（推荐，无竞争）
    GPIO_SetBits(GPIOB, GPIO_Pin_0);   // PB0 置高
    GPIO_ResetBits(GPIOB, GPIO_Pin_1); // PB1 拉低

    // 方式2：直接操作ODR寄存器
    // GPIOB->ODR |= (1<<0);
    // GPIOB->ODR &= ~(1<<1);
}
```

## 7. 按键触发RGBLED循环：红→绿→蓝，每次按键切换，单色亮1s
### 硬件约定
- 按键KEY1：PA0，上拉输入，下降沿触发（按下为低电平）
- 红灯：PB0、绿灯：PB1、蓝灯：PB2，低电平点亮（推挽输出）
```c
#include "stm32f10x.h"

// 延时函数，粗略1ms延时（72MHz系统时钟）
void Delay_ms(uint32_t ms)
{
    uint32_t i,j;
    for(i=0;i<ms;i++)
        for(j=0;j<7200;j++);
}

// GPIO初始化：PA0按键，PB0/PB1/PB2 RGB灯
void RGB_KEY_Init(void)
{
    GPIO_InitTypeDef GPIO_InitStruct;
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_GPIOB, ENABLE);

    // PA0 上拉输入（按键）
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_0;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_IPU;
    GPIO_Init(GPIOA, &GPIO_InitStruct);

    // PB0 PB1 PB2 推挽输出，初始全灭（高电平）
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_0 | GPIO_Pin_1 | GPIO_Pin_2;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_Out_PP;
    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_2MHz;
    GPIO_Init(GPIOB, &GPIO_InitStruct);
    GPIO_SetBits(GPIOB, GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2);
}

int main(void)
{
    RGB_KEY_Init();
    uint8_t color_state = 0; // 0红 1绿 2蓝
    while(1)
    {
        // 检测按键按下（PA0低电平）
        if(GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0) == 0)
        {
            while(GPIO_ReadInputDataBit(GPIOA, GPIO_Pin_0) == 0); // 等待松手消抖
            Delay_ms(20); // 软件消抖
            
            // 状态循环切换
            color_state = (color_state + 1) % 3;

            // 熄灭所有LED
            GPIO_SetBits(GPIOB, GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2);
            // 点亮对应颜色，保持1s
            switch(color_state)
            {
                case 0: GPIO_ResetBits(GPIOB, GPIO_Pin_0); break; // 红 PB0
                case 1: GPIO_ResetBits(GPIOB, GPIO_Pin_1); break; // 绿 PB1
                case 2: GPIO_ResetBits(GPIOB, GPIO_Pin_2); break; // 蓝 PB2
            }
            Delay_ms(1000);
            // 1s后熄灭
            GPIO_SetBits(GPIOB, GPIO_Pin_0|GPIO_Pin_1|GPIO_Pin_2);
        }
    }
}
```
### 逻辑说明
1. 每次按键松开后切换颜色状态，三色循环轮转；
2. 选中LED拉低点亮，固定保持1秒后熄灭，等待下一次按键触发；
3. 软件延时消抖，避免机械按键抖动误触发。

### 精简背诵考点总结
1. 必先开APB2时钟再初始化GPIO；
2. 输入三模式：浮空/上拉/下拉；输出四模式：推挽/开漏、复用推挽/开漏+模拟输入；
3. 输出速率：2M/10M/50M三档；
4. 电平控制优先BSRR/BRR原子操作，避免ODR读改写风险。
