# USART 串行异步通信 7道题完整解答
## 1. 串行异步通信数据格式、帧结构
### 帧组成（标准UART一帧）
异步通信以**字符帧**为传输单元，空闲电平为高电平，一帧依次包含：
1. **起始位**：1位，**低电平**，标志一帧开始，接收方检测到下降沿同步时钟
2. **数据位**：5~8位，低位先行，传输有效数据字节
3. **奇偶校验位（可选）**：0/1位，奇校验、偶校验，用于简单检错
4. **停止位**：1/1.5/2位，**高电平**，标志一帧结束，回到空闲状态

### 结构示意图（文本时序）
```
空闲(高) → 起始位(0) → D0 D1 D2 D3 D4 D5 D6 D7 → [校验位] → 停止位(1) → 空闲(高)
```
### 核心特点
收发双方**各自使用本地时钟**，仅靠起始位同步，不需要共享时钟线；波特率必须严格一致才能正确解析。

## 2. 波特率计算题
### 已知条件
帧格式：1起始位 + 8数据位 + 0校验位 + 1停止位
单个字符总位数：$1+8+0+1 = \boldsymbol{10\ \text{bit/字符}}$
每分钟传输字符数：9600 字符/分钟

### 计算
每秒字符数：
$$
N_{char} = \frac{9600}{60}=160\ \text{字符/秒}
$$
波特率（每秒比特数）：
$$
Baud = 160 \times 10 = \boldsymbol{1600\ \text{bps}}
$$

## 3. USART 工作原理
USART = 通用同步异步收发器，支持**异步UART**、同步时钟通信，核心分为发送通路、接收通路、波特率发生器三部分：
1. **波特率发生器**：对APB分频，生成收发双方匹配的移位时钟，控制每一位传输时长；
2. **发送端**：CPU写入数据到发送数据寄存器TDR，移入移位寄存器，在移位时钟下**低位先行**逐位输出；先发起始位、数据、校验、停止位，组帧后串行输出；发送完成置位TXE/TC标志，可触发中断；
3. **接收端**：持续检测RX引脚，检测到起始位下降沿后，用本地时钟采样每一位，逐位移入接收移位寄存器；整帧接收完成后，数据存入RDR，置位RXNE标志，校验错误、帧错误会置位对应状态位；
4. 同步模式：额外提供CK同步时钟，由主机输出时钟，从机跟随时钟严格同步传输。

---
## 4. USART 数据接收配置步骤（标准库）
1. 开启USART对应APB外设时钟、RX引脚GPIO时钟；
2. 配置RX引脚：浮空输入 / 上拉输入，复用为USART接收功能；
3. 初始化USART结构体：配置波特率、数据位长度、停止位个数、校验模式、异步模式、硬件流控关闭；
4. 调用`USART_Init()`载入基础配置；
5. 如需中断接收：使能`USART_IT_RXNE`接收非空中断，配置NVIC中断优先级、使能通道；
6. 使能USART外设：`USART_Cmd(USARTx, ENABLE)`；
7. 中断服务函数内读取`USART_ReceiveData()`获取接收字节，清除中断标志。

---
## 5. 全双工异步USART必备配置
全双工：独立TX发送、RX接收，可同时收发，配置要点：
1. 时钟：开启USARTx总线时钟、TX/RX对应GPIO端口时钟；
2. GPIO配置：
   - TX引脚：复用推挽输出；
   - RX引脚：浮空/上拉输入，复用接收；
3. 通信帧参数统一：波特率、数据位、停止位、奇偶校验，必须和对端设备完全一致；
4. 工作模式：选择**异步模式USART_Mode_USART**，关闭同步CK输出；
5. 收发使能：`USART_Mode_Rx | USART_Mode_Tx`同时开启收发；
6. 可选：开启RXNE接收中断、TC发送完成中断，或DMA收发；关闭硬件流控（无硬件握手时）；
7. 最后使能外设整体运行。

## 6. USART 初始化示例（STD库，USART1 PA9 TX PA10 RX）
```c
#include "stm32f10x.h"

void USART1_Init(uint32_t baudrate)
{
    GPIO_InitTypeDef GPIO_InitStruct;
    USART_InitTypeDef USART_InitStruct;
    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_USART1, ENABLE);

    // PA9 TX 复用推挽输出
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_9;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_AF_PP;
    GPIO_InitStruct.GPIO_Speed = GPIO_Speed_10MHz;
    GPIO_Init(GPIOA, &GPIO_InitStruct);

    // PA10 RX 浮空输入
    GPIO_InitStruct.GPIO_Pin = GPIO_Pin_10;
    GPIO_InitStruct.GPIO_Mode = GPIO_Mode_IN_FLOATING;
    GPIO_Init(GPIOA, &GPIO_InitStruct);

    // USART基础配置
    USART_InitStruct.USART_BaudRate = baudrate;
    USART_InitStruct.USART_WordLength = USART_WordLength_8b;
    USART_InitStruct.USART_StopBits = USART_StopBits_1;
    USART_InitStruct.USART_Parity = USART_Parity_No;
    USART_InitStruct.USART_HardwareFlowControl = USART_HardwareFlowControl_None;
    USART_InitStruct.USART_Mode = USART_Mode_Rx | USART_Mode_Tx; //全双工
    USART_Init(USART1, &USART_InitStruct);

    USART_Cmd(USART1, ENABLE);
}

//发送单字节
void USART1_SendData(uint8_t ch)
{
    while(USART_GetFlagStatus(USART1, USART_FLAG_TXE) == RESET);
    USART_SendData(USART1, ch);
}
```

## 7. USART 发送、接收中断事件分类
### 一、发送相关中断（3类核心）
1. **TXE（发送数据寄存器空中断）**：TDR数据移入移位寄存器，寄存器为空，可写入下一个待发字节，连续发送用；
2. **TC（发送完成中断）**：整帧数据全部移位发送完毕，总线回到空闲，一帧完整发送结束；
3. **CTS中断**：硬件流控CTS引脚电平变化，硬件握手允许/禁止发送。

### 二、接收相关中断（4类核心）
1. **RXNE（接收非空中断）**：成功接收一帧数据，RDR存有有效数据，最常用接收中断；
2. **ORE（溢出错误中断）**：新数据到来前旧数据未读取，数据覆盖溢出报错；
3. **PE（奇偶校验错误中断）**：接收数据奇偶校验不匹配，数据传输出错；
4. **FE（帧格式错误中断）**：停止位缺失、帧结构非法，起始/停止位解析错误。

---
### 背诵精简考点
1. 异步帧：起始位+数据位+校验位+停止位，空闲高电平，起始位下降沿同步；
2. 波特率=每帧总位数 × 每秒字符数；
3. 全双工必须同时配置TX复用输出、RX输入，收发同时使能；
4. 发送核心中断：TXE、TC；接收核心中断：RXNE，另有校验、帧错误、溢出错误中断。
