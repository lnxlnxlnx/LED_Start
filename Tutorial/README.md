# XinYueC + 设计模式 学习路线

这套教程面向你当前项目（STM32 + XinYueC），目标是把“设计模式”变成可落地的嵌入式代码能力，而不是只停留在概念。

## 阅读顺序

1. `01_课程总览.md`
2. `02_12周课程大纲.md`
3. `03_设计模式与XinYue映射.md`
4. `04_里程碑与实战项目.md`
5. `05_23种设计模式嵌入式落地手册.md`
6. `06_从需求到代码_模式落地模板.md`

## 使用方式

- 每周完成一个主题，必须包含：阅读、编码、复盘。
- 所有练习优先在当前仓库中完成，不做脱离工程的“玩具代码”。
- 每个阶段都要产出可运行结果（串口日志、行为验证、代码 diff）。
